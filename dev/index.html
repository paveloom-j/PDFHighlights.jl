PDFHighlights.jl

Export highlights from PDF files to a CSV file with one call.

Code Coverage Repository & License

A package for exporting highlights (and related data) from PDF files and saving them to a CSV.

Package Features

  • Import highlights into a new or already created CSV file with one call
  • Concatenate highlights utilizing comments
  • Extract any piece of a highlight from a PDF or CSV file
